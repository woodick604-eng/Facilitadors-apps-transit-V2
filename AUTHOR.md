## Descripció

Plataforma centralitzada per a la gestió d'atenees i informes fotogràfics destinada a agents de patrulla. Inclou sistema d'autenticació, dashboard d'administrador, gestió d'aplicacions, sistema de dictats amb IA (Google Gemini), registre d'accions i control d'accés per rol.

## Tecnologies

- React 19 + TypeScript
- Vite 6
- Firebase / Firestore
- Google Generative AI (Gemini)
- Tailwind CSS
